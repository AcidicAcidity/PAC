# Jira analog in WEB format

My project call's PAC.

PAC - Plan. Assign. Collab. It's own tagline.

## What YOU can do in owr project?

    1. You can create and publish your tasks to all department or you can assign tasks to another user.
    2. You can track public tasks in your department.
    3. You can use different color schemes to customize your work placement.
    4. You can combine into collabs to trak common tasks.
    5. You can enable VIM-like input mode to navigate using the hjkl buttons.
